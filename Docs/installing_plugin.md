# Install the Complexity Plug

## Follow closely the instructions in the screenshots (below)

![Click Help, then Install New Software ...](images/InstallN01.png)

![Click the Add button](images/InstallN02.png)

![Click the Archive button](images/InstallN03.png)

![Naviage to csse220/Administration/EclipsePlugin folder, Select ComplexityPluginArchive, Click Open](images/InstallN04.png)

![Enter the name ComplexityPlugin, then click Add](images/InstallN05.png)

![Check the Elegance checkbox, then click Next](images/InstallN06.png)

![Click Finish](images/InstallN07.png)

![Check the Unsigned checkbox, then click Trust Selected](images/InstallN08.png)

![Click Restart Now](images/InstallN09.png)

## That's it! You've installed the Complexity Plugin

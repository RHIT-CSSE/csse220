
Import these .zip Eclipse projects into Eclipse to see examples of the ArrayList operations listed below

Two Eclipse project .zip's
   ArrayListPowerOpsV1.zip
   ArrayListPowerOpsV2.zip

Java ArrayList "Power" operations:
   addAll - takes parameter from Collections interface
   forEach - takes parameter from Consumer interface
   removeIf - takes parameter from Predicate interface
   removeAll - takes parameter from Collections interface
   replaceAll - takes parameter from UnaryOperator interface
   sort - takes parameter from Comparator interface
